# Resume Webpage

This project is a personal resume converted into a webpage using HTML, CSS, and JavaScript.

## Features
- Dark mode toggle
- Show more / show less projects
- Contact form with validation
- Download resume option

## Technologies Used
- HTML
- CSS
- JavaScript

## Description
I created this webpage to represent my resume in an interactive and user-friendly way. 
It includes basic styling and simple JavaScript functionality to improve user experience.

## How to Run
1. Download the project files
2. Open resume.html in any browser
